FLEXBOX

就是box里面的元素，通过flex来操作。解决垂直居中，还有并排同高。

又一个main axis 和一个 cross axis（对齐用的）

FLEX CONTAINER

  • gap: 0 | { length }
  • justify-content: flex-start | flex-end | center | space-between | space around | space-evenly 通过横轴排列元素
  • align-items: stretch | flex-start | flex-end | center | baseline 通过纵轴排列元素
  • flex-direction: row | row-reverse | column | column-reverse
  • flex-wrap: nowrap | wrap | wrap-reverse
  • align-content: stretch | flex-start | flex-end | center | space-between | space-around

FLEX ITEMS

  • align-self: auto | stretch | flex-start | flex-end | center | baseline
  • flex-grow: 0 | { integer } 1为开启，自动填充剩余空间，如果有的话(常用于按比例填充)
  • flex-shrink: 1 | { integer } 1为开启， flex 元素仅在默认宽度之和大于容器的时候才会发生收缩，其收缩的大小是依据flex-shrink 的值
  • flex-basis: auto | { length } 并非严格宽度，在不超过的情况下是设置的大小
  • flex: 0 1 auto | { int } { int } { len } 第一个是flex-grow 第二个是flex-shrink 第三个是flex-basis
  • order: 0 | { integer } 默认初始都为0
